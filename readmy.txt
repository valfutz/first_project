Описание:
TodoListApp представляет собой простое приложение для управления списком задач и подзадач с использованием графического интерфейса пользователя (GUI) на основе библиотеки Tkinter.

Основные возможности:

    Открытие и сохранение списка задач:
        Пользователь может открывать существующие списки задач из файла.
        Список задач сохраняется в файл после внесения изменений.

    Добавление задач:
        Пользователь может добавлять новые задачи с указанием срока выполнения (deadline).
        Возможность добавления подзадач для каждой задачи.

    Управление сроками и завершение задач:
        Возможность изменения срока выполнения задачи.
        Пометка задачи как выполненной.

    Удаление и очистка:
        Возможность удаления как основных задач, так и подзадач.
        Возможность очистки всего списка задач.

Интерфейс:

    Открытие файла:
        Кнопка "Open file" позволяет пользователю выбирать файл для загрузки списка задач.
        Поле ввода "File path" отображает путь выбранного файла.

    Добавление задач:
        Поля ввода для текста задачи и срока выполнения.
        Кнопка "Add" добавляет новую задачу в список.

    Управление сроками:
        Кнопки "Choose deadline" и "Change deadline" позволяют выбрать и изменить срок выполнения задачи.

    Отметка задачи как выполненной:
        Кнопка "Done" помечает выбранную задачу как выполненную.

    Удаление задач:
        Кнопка "Delete" удаляет выбранную задачу или подзадачу.

    Очистка списка:
        Кнопка "Clear all" полностью очищает список задач.

Внешние зависимости:

    Используется библиотека Tkinter для создания графического интерфейса.
    Встроенные модули json, datetime, os, filedialog, messagebox используются для работы с данными и файлами.

Структура кода:

    Код разделен на методы и функции для облегчения чтения и понимания.
    Используется пользовательская библиотека customtkinter (ctk) для стилизации виджетов Tkinter.

Запуск приложения:

    Если файл запускается напрямую (не импортируется), создается экземпляр TodoListApp и вызывается его метод mainloop() для отображения GUI.

Расширение и внесение вклада:

    Проект может быть расширен добавлением новых функций или улучшен согласно потребностям пользователя.
    Для внесения вклада в проект предусмотрены методы add_task, delete_task, mark_as_done и др.

Лицензия:

    Отсутствует указание конкретной лицензии в коде. Рекомендуется добавить явное уведомление о лицензии для других разработчиков.
